tor-static-builds
=================

Description
------------
The great ["tor"](https://www.torproject.org/download/tor/) tool, from the TOR project.

The building workflow is [here](https://github.com/mycodedoesnotcompile2/tor-static-builds/blob/main/.github/workflows/main.yml).

See the [Github release page](https://github.com/mycodedoesnotcompile2/tor-static-builds/releases) of this repository to access prebuilt binaries for different architectures (x64, arm64) and different libc (GNU, musl).
